== Description ==

Disable Settings For WP is a WordPress plugin that allows you to disable right-click functionality and hide the admin bar in your WordPress site. This plugin provides an easy way to customize the user experience on your website by preventing unwanted actions and improving security.

== Features ==

- Disable right-click functionality on your site.
- Hide the WordPress admin bar for specific user roles.

== Installation ==

1. Upload the `disable-settings-for-wp` folder to the `/wp-content/plugins/` directory or install it directly from the WordPress Plugin Repository.
   
   **Option 1:** Manual Installation
   - Download the plugin ZIP file from the WordPress Plugin Repository.
   - Extract the contents of the ZIP file.
   - Upload the `disable-settings-for-wp` folder to the `/wp-content/plugins/` directory using FTP or your hosting provider's file manager.

   **Option 2:** Installation via WordPress Dashboard (Recommended)
   - Go to your WordPress Dashboard.
   - Navigate to "Plugins" > "Add New."
   - Search for "Disable Settings For WP."
   - Click "Install Now" and then activate the plugin.

2. After installation, go to the "Disable Settings" menu under "Settings" in your WordPress dashboard to configure the plugin settings.

== Usage ==

1. Navigate to "Settings" > "Disable Settings" in your WordPress dashboard.
2. Configure the options according to your preferences:
   - Enable Right-Click Disabling: Check this option to disable right-click functionality on your site.
   - Hide Topbar Panel: Check this option to hide the WordPress admin bar.
3. Click the "Save Changes" button to save your settings.

== Frequently Asked Questions ==

= How can I restore right-click functionality on my site?

To restore right-click functionality, simply uncheck the "Enable Right-Click Disabling" option in the plugin settings and save your changes.

= How can I bring back the WordPress admin bar?

If you have hidden the admin bar and want to show it again, uncheck the "Hide Topbar Panel" option in the plugin settings and save your changes.
